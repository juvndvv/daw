Elementos GRID

Definiendo áreas

Un Grid Layout debe tener un elemento padre con la propiedad CSS display establecida a grid o inline-grid.

Los elementos hijos directos del contenedor de rejilla se convierten automáticamente en elementos de rejilla.

Propiedades GRID

Contenedor GRID

  • display: grid
  • grid-template-areas
  • grid-template-columns
  • grid-template-rows

GRID items (hijos directos)

  • grid-area
